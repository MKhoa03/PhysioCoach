## Installation nach dem Klonen
Nach dem Klonen des Projekts müssen die Abhängigkeiten für das **Frontend** installiert werden. Dafür in den "Frontend"-Ordner wechseln und "npm install" ausführen:

```bash
cd frontend
npm install
```

> **Hinweis:** Eventuell muss man Visual Studio Code schließen und wieder öffnen.

## App starten
Um die App zu starten, in den Frontend-Ordner  und "npx expo start" ausführen:

```bash
npx expo start
```

> **Hinweis:** Um den Cache zu leeren, im Frontend-Ordner ausführen:  
> ```bash
> npx expo start -c
> ```
